# Support

Use the repository issue tracker for project-specific questions, bugs, and feature requests.

## Where to ask

- Bugs: open an issue in the affected repository.
- Feature requests: open an issue in the repository that would own the work.
- Documentation issues: open an issue in the related docs or project repository.
- Security reports: follow [SECURITY.md](SECURITY.md).
- General project discovery: start from the [Sevoniva organization profile](profile/README.md).

## 中文

具体项目的问题、缺陷和需求，请优先在对应仓库提交 issue。

适合的入口：

- 缺陷：在受影响仓库提交 issue。
- 功能需求：在负责该能力的仓库提交 issue。
- 文档问题：在相关文档或项目仓库提交 issue。
- 安全问题：查看 [SECURITY.md](SECURITY.md)。
- 项目查找：从 [Sevoniva 组织首页](profile/README.md) 开始。
