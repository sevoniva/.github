# Sevoniva

[English](#english) | [中文](#中文)

## English

Sevoniva is a public catalog for data platforms and engineering tools. It groups open repositories by use case so visitors can find MCP servers, Codex skills, BI projects, workflow platforms, and documentation from the GitHub organization page.

This catalog lists public repositories only.

### Start here

| Area | Repository | Use case |
| --- | --- | --- |
| MCP server | [oceanbase-mcp](https://github.com/sevoniva/oceanbase-mcp) | OceanBase metadata discovery and read-only SQL access through MCP. |
| Codex skills | [skills](https://github.com/sevoniva/skills) | Reusable Codex skills for local development and automation workflows. |
| BI platform | [Crest](https://github.com/sevoniva/Crest) | Datasets, dashboards, reports, and BI project work. |
| ChatBI | [cogniflow](https://github.com/sevoniva/cogniflow) | SQL generation, chart rendering, and conversational analytics. |
| Workflow platform | [Koravo](https://github.com/sevoniva/Koravo) | Process automation and data orchestration based on Flowable. |
| Delivery platform | [Nivora](https://github.com/sevoniva/Nivora) | CI/CD, GitOps, deployment, artifacts, and release audit. |

## 中文

Sevoniva 是一个公开项目目录，聚合数据平台与工程工具相关仓库。这里按用途整理仓库，方便从 GitHub 组织首页直接找到 MCP 服务、Codex skills、BI 项目、流程平台、交付工具和文档入口。

本目录只展示公开仓库。

### 快速入口

| 分类 | 仓库 | 用途 |
| --- | --- | --- |
| MCP 服务 | [oceanbase-mcp](https://github.com/sevoniva/oceanbase-mcp) | 通过 MCP 访问 OceanBase 元数据和只读 SQL 查询。 |
| Codex skills | [skills](https://github.com/sevoniva/skills) | 面向本地开发和自动化工作流的 Codex skills。 |
| BI 平台 | [Crest](https://github.com/sevoniva/Crest) | 数据集、仪表盘、报表和 BI 项目工作。 |
| ChatBI | [cogniflow](https://github.com/sevoniva/cogniflow) | SQL 生成、图表渲染和对话式分析。 |
| 流程平台 | [Koravo](https://github.com/sevoniva/Koravo) | 基于 Flowable 的流程自动化和数据编排。 |
| 交付平台 | [Nivora](https://github.com/sevoniva/Nivora) | CI/CD、GitOps、部署、制品和发布审计。 |
