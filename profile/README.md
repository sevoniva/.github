# Sevoniva

Sevoniva builds data products, developer infrastructure, and workflow tools for teams that need clear operations, governed access, and practical delivery control.

## Projects

### Data and analytics

- [Crest](https://github.com/sevoniva/Crest) - BI platform for datasets, dashboards, reports, and governed data access.
- [crest-core](https://github.com/sevoniva/crest-core) - Minimal Crest distribution for core BI and reporting workflows.
- [cogniflow](https://github.com/sevoniva/cogniflow) - ChatBI service for governed SQL, chart rendering, and conversational analytics.
- [oceanbase-mcp](https://github.com/sevoniva/oceanbase-mcp) - Read-only MCP server for OceanBase metadata discovery and governed SQL access.

### Workflow and delivery

- [Koravo](https://github.com/sevoniva/Koravo) - Process and data orchestration platform built on Flowable.
- [Nivora](https://github.com/sevoniva/Nivora) - DevOps delivery control plane for CI/CD, GitOps, deployment, artifacts, and release audit.
- [skills](https://github.com/sevoniva/skills) - Reusable Codex and agent skills for local automation workflows.

### Product experiments

- [Memorica](https://github.com/sevoniva/Memorica) - Knowledge and memory service for notes, recall, and structured personal records.
- [NivaHome](https://github.com/sevoniva/NivaHome) - Smart home platform for devices, automation, spaces, and local services.
- [Crest-iOS](https://github.com/sevoniva/Crest-iOS) - iOS client for Crest dashboards and reports.
