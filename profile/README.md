# Sevoniva

[English](#english) | [中文](#中文)

## English

Sevoniva is a project portfolio for data products and engineering infrastructure. It includes BI and reporting systems, database integration tools, workflow platforms, delivery tools, and local automation utilities.

### Main areas

- BI, dashboards, reports, and data visualization.
- SQL tools, OceanBase integration, and metadata services.
- Workflow orchestration, BPMN, and process automation.
- CI/CD, GitOps, deployment, release audit, and developer tooling.
- Local automation and personal productivity tools.

### Projects

#### Data platforms

- [Crest](https://github.com/sevoniva/Crest) - BI platform for datasets, dashboards, reports, and dataset management.
- [crest-core](https://github.com/sevoniva/crest-core) - Compact Crest distribution for BI and reporting workflows.
- [cogniflow](https://github.com/sevoniva/cogniflow) - ChatBI service for SQL generation, chart rendering, and conversational analytics.
- [oceanbase-mcp](https://github.com/sevoniva/oceanbase-mcp) - Read-only MCP server for OceanBase metadata discovery and SQL access.

#### Workflow and delivery platforms

- [Koravo](https://github.com/sevoniva/Koravo) - Process and data orchestration platform built on Flowable.
- [Nivora](https://github.com/sevoniva/Nivora) - DevOps delivery control plane for CI/CD, GitOps, deployment, artifacts, and release audit.
- [skills](https://github.com/sevoniva/skills) - Reusable Codex skills for local development and automation workflows.

#### Applications and tools

- [Memorica](https://github.com/sevoniva/Memorica) - Knowledge and memory service for notes, recall, and structured personal records.
- [NivaHome](https://github.com/sevoniva/NivaHome) - Smart home platform for devices, automation, spaces, and local services.
- [Crest-iOS](https://github.com/sevoniva/Crest-iOS) - iOS client for Crest dashboards and reports.

## 中文

Sevoniva 围绕数据产品与工程基础设施建设项目矩阵，覆盖 BI 报表、数据库集成、流程平台、研发交付和本地自动化工具。

### 主要方向

- BI、仪表盘、报表和数据可视化。
- SQL 工具、OceanBase 集成和元数据服务。
- 流程编排、BPMN 和流程自动化。
- CI/CD、GitOps、部署、发布审计和研发工具。
- 本地自动化与个人效率工具。

### 项目

#### 数据平台

- [Crest](https://github.com/sevoniva/Crest) - 面向数据集、仪表盘、报表和数据权限场景的 BI 平台。
- [crest-core](https://github.com/sevoniva/crest-core) - 面向 BI 与报表场景的 Crest 精简发行版。
- [cogniflow](https://github.com/sevoniva/cogniflow) - 面向 SQL 生成、图表渲染和对话式分析的 ChatBI 服务。
- [oceanbase-mcp](https://github.com/sevoniva/oceanbase-mcp) - 面向 OceanBase 元数据发现和 SQL 查询的只读 MCP 服务。

#### 流程与交付平台

- [Koravo](https://github.com/sevoniva/Koravo) - 基于 Flowable 的流程与数据编排平台。
- [Nivora](https://github.com/sevoniva/Nivora) - 面向 CI/CD、GitOps、部署、制品和发布审计的 DevOps 交付控制平台。
- [skills](https://github.com/sevoniva/skills) - 面向本地开发和自动化工作流的 Codex skills。

#### 应用与工具

- [Memorica](https://github.com/sevoniva/Memorica) - 面向笔记、回忆和结构化个人记录的知识与记忆服务。
- [NivaHome](https://github.com/sevoniva/NivaHome) - 面向设备、自动化、空间和本地服务的智能家居平台。
- [Crest-iOS](https://github.com/sevoniva/Crest-iOS) - 面向 Crest 仪表盘和报表的 iOS 客户端。
